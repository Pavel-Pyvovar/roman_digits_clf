About Roman Digits Classifier

The Roman Digits Classifier is an application developed to demonstrate how Machine Learning can be useful. It solves a supervised learning problem.

The model takes as an input a picture of roman digit from one to eight and gives as an output a predicted class of an image.

At the core of this system lies a Convolutional Neural Net. It is a widely used Deep Learning model which often applied for the task of image classification.
